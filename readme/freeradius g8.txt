client router {
    ipaddr = <路由器的IP地址>
    secret = <共享密钥>
    nas_type = other
}

WPA algorithms: CCMP-128(AES)
AuthServer Address: <宿主机ip或服务器ip>
Auth Server Port 1812
Auth Shared Secret: <共享密钥>
